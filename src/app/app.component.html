Needlework Canvas Creator

A handy tool for creating a needlework canvas from an image.

For who is this tool for?
If you are interested in needlework and you would like to create a model based on an image for your artwork.

All you need is just to upload the image. Then you can adjust the size of the canvas or the number of used colors.
